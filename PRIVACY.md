# Privacy Policy for Anti Ultra

Last updated: March 13, 2026

Anti Ultra is a browser extension that hides Google AI Ultra upsell elements on `https://gemini.google.com/*`.

## What data Anti Ultra collects

Anti Ultra does not collect, sell, or transfer personal information.

Anti Ultra only stores one browser setting using extension storage:

- whether the extension is currently enabled or disabled

## What Anti Ultra does not do

Anti Ultra does not:

- collect personally identifiable information
- collect financial, health, authentication, or communication data
- sell or transfer user data to third parties
- send browsing activity or Gemini page content to external servers
- use analytics, advertising trackers, or remote code

## Website access

Anti Ultra runs only on `https://gemini.google.com/*` so it can identify and hide specific Google AI Ultra upsell interface elements on that site.

The extension processes page content locally in the browser only to detect those targeted upsell elements. That processing is not used to build profiles, track users, or transmit page content elsewhere.

## Third-party links

The popup includes an optional support link to Buy Me a Coffee. If you click that link, you will leave the extension and visit a third-party website that operates under its own terms and privacy policy.

## Changes

If this privacy policy changes, the updated version will be published in this repository.
